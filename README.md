## 🌟 What Is Cherry-101?

**Cherry-101** is the very first program ever written in [**CherryScript**](https://github.com/Infinite-Networker/CherryScript) — a pioneering domain-specific language designed from the ground up at **Cherry Computer Ltd.** to simplify and supercharge AI & machine learning development.

In one clean, streamlined language, Cherry-101 demonstrates a **complete ML pipeline** — from raw data queries all the way through to model training, deployment, and live predictions. No boilerplate. No patchwork of frameworks. Just CherryScript.

This project proves that domain-specific languages aren't just a novelty — they are the *future* of AI development tooling.

## 📄 Quick-Start Example

Here's what a full ML pipeline looks like in **CherryScript** — the language Cherry-101 was written in:

```javascript
// 🍒 Cherry-101 — Your First CherryScript ML Pipeline

// 1️⃣  Connect to your database
var db = connect("mysql://localhost:3306/ecommerce_db", "admin", "password")

// 2️⃣  Pull your training & test data
var training_data = db.query("SELECT * FROM orders_train WHERE status = 'completed'")
var test_data     = db.query("SELECT * FROM orders_test")

// 3️⃣  Prepare H2O data frames
var training_frame = h2o.frame(training_data)
var test_frame     = h2o.frame(test_data)

// 4️⃣  Train with AutoML — CherryScript picks the best model for you
var ml_model = h2o.automl(training_frame, "is_return")

// 5️⃣  Deploy to a live REST endpoint — one line, zero fuss
var api = deploy(ml_model, "http://localhost:8080/predict")

// 6️⃣  Predict & evaluate
var predictions = predict(ml_model, test_frame)
var metrics     = evaluate(predictions, "is_return", "results.json")

print("✅ Pipeline complete! Accuracy: ${metrics.accuracy * 100}%")
```

> **That's it.** Database → AutoML → Deployment → Evaluation. In one script. In one language.

## 🛠️ Installation & Setup

### Prerequisites

- Python **3.8+**
- A running **MySQL** instance *(for database adapter)*
- **H2O** cluster *(for AutoML features)*

### 1. Clone the Repository

```bash
git clone https://github.com/Infinite-Networker/Cherry-101.git
cd Cherry-101
```

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

### 3. Run the Example Pipeline

```bash
python src/Cherry-101-cli.py examples/test.cherry-101
```
